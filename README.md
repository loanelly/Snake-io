# 🐍 Snake-IO (Simple Version)

Упрощенная, высокооптимизированная и стабильная версия классической игры «Змейка» на Python и Pygame. Данная сборка лишена тяжелых графических элементов, меню и задержек, что делает её идеальным выбором для интеграции в проекты машинного обучения.

---

## 🤖 Назначение: Машинное обучение (Machine Learning)

Этот скрипт специально разработан как изолированная среда для **обучения с подкреплением (Reinforcement Learning)**. Чистый и понятный код позволяет легко превратить игру в кастомное окружение (по подобию OpenAI Gym / Gymnasium).

### 🌟 Почему эта версия идеальна для ИИ:
- **Мгновенный сброс (Instant Reset):** При столкновении со стеной или хвостом игра не зависает на экране «Game Over» и не требует нажатия клавиш. Метод `return` мгновенно перезапускает игровой цикл, позволяя агенту ИИ безболезненно начинать новые эпохи (episodes) обучения.
- **Вынесенный интерфейс:** Панель счета полностью вынесена за пределы игрового поля (`HEADER_HEIGHT`). Это позволяет нейросети считывать чистую матрицу игрового поля (массив пикселей или сетку координат) без визуального шума от текста.
- **Двойное управление:** Вшитая поддержка WASD и Стрелочек позволяет легко симулировать действия агента через генерацию виртуальных нажатий клавиш (`pygame.event.post`).
- **Легковесность:** Минимальное количество зависимостей и отсутствие сторонних шрифтов гарантируют максимальную скорость рендеринга и вычислений.

## ⌨️ Управление (Для человека)
Если вы запускаете игру вручную, управление дублируется на две раскладки:
- **Стрелочки:** `↑`, `↓`, `←`, `→`
- **WASD:** `W`, `A`, `S`, `D`

## 🚀 Быстрый старт

### 1. Установка зависимостей
Для работы окружения необходима только библиотека Pygame:
```bash
pip install pygame
```

### 2. Запуск игры
```bash
python Snake-io-simple-version.py
```
