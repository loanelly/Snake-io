# Snake-io
Snake-IO — классическая аркадная змейка на Python и Pygame. Игра включает два режима: расслабленный «Классический» и хардкорный с динамическим ростом уровней и скорости. Доступна мгновенная смена биндов управления в меню на выбор: Стрелочки, WASD или IJKL. Код оптимизирован против зависаний буфера окон Windows.
