# 🐍 Snake-IO

Классическая аркадная игра «Змейка» на Python с использованием библиотеки **Pygame**. Проект разработан с упором на стабильность: архитектура полностью защищена от зависаний графического буфера Windows и багов «черного экрана».

---

## 🌟 Особенности проекта

- **100% Кроссплатформенность:** Полный отказ от системных шрифтов Windows (`SysFont`) в пользу встроенных шрифтов Pygame. Игра гарантированно отображает текст и запускается на любой ОС (Windows, macOS, Linux).
- **Линейная архитектура:** Код избавлен от вложенных фоновых циклов на экране проигрыша. Это исключает баги с блокировкой системных событий (`pygame.event.get()`) и зависанием процесса в диспетчере задач.
- **Динамические настройки:** Изменение конфигурации клавиш происходит «на лету» прямо из графического интерфейса.

---

## 🕹 Игровые режимы

1. **Classic Mode (Классика):** Игра на фиксированной комфортной скорости. Отличный вариант для расслабленной игры и ленивого набора очков.
2. **Levels Mode (Уровни):** Динамический режим для хардкорных игроков. Каждые 3 съеденных яблока повышают уровень сложности и увеличивают скорость движения змейки.

---

## ⌨️ Настройка управления (Бинды)

В меню настроек (клавиша **3**) вы можете мгновенно переключить управление на один из трех популярных пресетов:
- **ARROWS:** Стандартные стрелочки клавиатуры (`↑`, `↓`, `←`, `→`).
- **WASD:** Классическая геймерская раскладка (`W`, `S`, `A`, `D`).
- **IJKL:** Альтернативная раскладка для левшей или игры двумя руками (`I`, `K`, `J`, `L`).

*В игре реализована защита от разворота на 180 градусов (вы физически не сможете врезаться в себя, случайно нажав кнопку назад).*

## 🚀 Быстрый старт

### 1. Требования
Для запуска игры необходим установленный Python версии 3.10 или выше, а также библиотека Pygame.

### 2. Установка зависимостей
Откройте терминал/командную строку и установите Pygame:
```bash
pip install pygame
```

### 3. Запуск
Запустите файл игры напрямую из папки **двойным кликом мыши** или через терминал:
```bash
python Snake-io.py
```

## 🎮 Навигация по меню
- **В главном меню:** Нажмите `1`, `2` или `3` для выбора режима или перехода в настройки.
- **В настройках:** Нажмите `1`, `2` или `3` для выбора раскладки. Нажмите `ESC` для возврата в меню.
- **В игре:** Нажмите `ESC` для моментального проигрыша и выхода.
- **На экране Game Over:** Игра покажет ваш финальный счет, подождет 3 секунды и автоматически вернет вас в главное меню.
